# Artistic Grid Mode

## Description

Independent foreground/background color palettes for creating artistic patterns in the contrast grid, plus an "art mode" view and color blending options.

### Core Features

1. **Separate FG/BG Palettes**: Allow users to specify different color sets for rows (foreground) and columns (background), enabling asymmetric grids and true 2D pattern design

2. **Art Mode**: Remove text labels and WCAG compliance information to display the grid as pure visual art

3. **Color Blending**: Option to blend/mix the foreground and background colors in each cell to create additional visual effects

### Use Cases

- Creating artistic patterns and geometric designs using color contrast
- Generating shareable color art URLs
- Exploring creative color relationships beyond accessibility testing
- Educational tool for understanding color interaction and mixing

### URL Structure

```
?fg=FF0000,00FF00,0000FF&bg=000000,FFFFFF,808080
?art=true  # Enable art mode (remove labels)
?blend=true  # Enable color blending
```
